# Hospital Readmissions Dashboard

This project is an **interactive dashboard** built with **Streamlit** and **Plotly** for visualizing and analyzing hospital readmission data. It allows users to explore patient statistics, variable distributions, and factors affecting length of stay using **interactive charts** and metrics.

---

## Features
- **Key Metrics**: Total patients, average stay, readmission rate, average procedures, lab tests, and medications.  
- **Variable Distribution by Age**: Histogram or pie chart for different variables filtered by age group.  
- **Factors Influencing Length of Stay**: Scatter plots showing relationships between medical interventions and hospitalization time.  
- **Length of Stay by Primary Diagnosis**: Box plots showing distribution based on diagnosis and readmission status.  
- **Medical Interventions by Age and Readmission**: Bar charts showing average medical procedures grouped by age and readmission status.
